# Как добавить навык

## Структура навыка

Создайте папку в `skills/`:

```text
skills/my-new-skill/
├── SKILL.md    ← Обязательно. Инструкция для Claude
└── README.md   ← Обязательно. Документация для человека
```

## SKILL.md

Файл с YAML frontmatter и markdown-инструкциями:

```yaml
---
name: my-new-skill
description: |
  Краткое описание что делает навык (1-3 предложения).

  USE THIS SKILL when the user wants to:
  - Перечислите триггерные фразы
  - Чем больше вариантов — тем надёжнее срабатывание
---

# Название навыка

Подробные пошаговые инструкции для Claude...
```

**Правила для description:**

- Первые 1-3 предложения — что делает навык
- Блок `USE THIS SKILL when` — конкретные фразы пользователя
- Добавьте фразы на русском если навык для RU-рынка

## README.md

Документация для человека:

- Что это и зачем
- Как установить (3 варианта: в проект, глобально, через CLAUDE.md)
- Что создаётся (таблица файлов)
- Требования и зависимости
- Схема архитектуры

## Чеклист перед коммитом

- [ ] `SKILL.md` содержит YAML frontmatter с `name` и `description`
- [ ] `README.md` содержит инструкцию по установке
- [ ] Навык добавлен в таблицу в корневом `README.md`
- [ ] Нет секретов, токенов, паролей в файлах навыка
